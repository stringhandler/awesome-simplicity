# Contributing to Awesome Simplicity

Thanks for wanting to contribute!

## Adding an item

1. Fork this repo and create a branch.
2. Add your item to the relevant section in `README.md`.
3. Format it as: `- [Name](URL) - Short description ending with a period.`
4. Keep descriptions concise â€” one sentence is ideal.
5. Open a pull request with a brief explanation of why the resource is awesome.

## Guidelines

- Links must be working and publicly accessible.
- Projects should be actively maintained or historically significant.
- No self-promotion without meaningful community adoption.
- Prefer official or canonical sources over mirrors.

## Sections

If your item doesn't fit an existing section, feel free to propose a new one in your PR.
